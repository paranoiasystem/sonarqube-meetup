## Non utilizzare "pippo" come nome per variabili o costanti.

### Motivazione

Il nome "pippo" è un nome generico che non descrive il contenuto della variabile o costante.

### Soluzione

Scegliere un nome più descrittivo.

### Esempio

```javascript
// Bad
const pippo = 1;
```

```javascript
// Good
const numero = 1;
```
